## Technológiai stack
- **Frontend**: React, TypeScript, Vite
- **Backend**: .NET 8 ASP.NET Core Web API
- **Adatbázis**: Entity Framework Core, SQLite

Az adatbázist automatikusan generálja, de szerepel a fájlok között egy teszt adatbázis is.

## Futtatás:
- **Backend**: Visual Studio start gomb, vagy "dotnet run" parancs.
- **Frontend**: "npm install" majd "npm run dev", http://localhost:5173/ címen elérhető (vagy amit a konzol kiír)
